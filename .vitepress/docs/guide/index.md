# Getting started

## Installation

### Prerequisites

- Node.js version 18 or higher.
- A package manager: e.g. npm, pnpm, ...

::: code-group

```sh [npm]
$ npm add -D @ogs-gmbh/ngx-http
```

```sh [pnpm]
$ pnpm add -D @ogs-gmbh/ngx-http
```

```sh [yarn]
$ yarn add -D @ogs-gmbh/ngx-http
```

```sh [bun]
$ bun add -D @ogs-gmbh/ngx-http
```

:::

Please follow our specific guides to understand and unlock the full potential.
